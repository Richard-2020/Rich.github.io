Introduction To CSS

CSS Stands for Cascading Style Sheet.
CSS describes how HTML elements are to be displayed in paper or in other media. CSS can be inserted in the html document from the html head element , Inline the html element or using an external style sheet.

CSS Syntax

body{ font-size:20px; }
In the above declaration, body is the selector, font-size is the property and 20px is the value.

Element Selectors

You can select all p elements with this code
p {
color : blue ;
}
This will select each and every < p > element found in a webpage.
CSS is also used to define styles for your web pages including the design layout and variations display for different screen sizes.

The Id Selector

In CSS the html id attribute value can be used to format the element using CSS.
The Id uses the # before the selector in css .This tells the browser that this is a html attribute.
For Example :
# para{
color : red ;
}
In HTML : < p id= " para " >

The Class Selector


In CSS the (.) represents the class attribute in html.
For Example :
.para{
color:white;
}
Represents this in html : < p class = " para ">
