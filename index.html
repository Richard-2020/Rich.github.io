Introduction To CSS

CSS Stands for Cascading Style Sheet.
CSS describes how HTML elements are to be displayed in paper or in other media. CSS can be inserted in the html document from the html head element , Inline the html element or using an external style sheet.

CSS Syntax

body{ font-size:20px; }


In the above declaration, body is the selector, font-size is the property and 20px is the value.

Element Selectors

You can select all p elements with this code
p {
color : blue ;
}
This will select each and every < p > element found in a webpage.
CSS is also used to define styles for your web pages including the design layout and variations display for different screen sizes.

The Id Selector

In CSS the html id attribute value can be used to format the element using CSS.
The Id uses the # before the selector in css .This tells the browser that this is a html attribute.
For Example :
# para{
color : red ;
}
In HTML : < p id= " para " >

The Class Selector


In CSS the (.) represents the class attribute in html.
For Example :
.para{
color:white;
}
Represents this in html : < p class = " para ">

Css Comments

Comments are used to explain code and may help you edit the code at a later date. A single line comment in css is written with a // before the comment while multiline comments are written with a /* and */.

Css Styles

 There are 3 ways to insert Css in a html document.

1. External stylesheet

2.Internal stylesheet

3.Inline stylesheet

External Stylesheet

With an external stylesheet,web developers can insert css in a html document. We do it with < link > element ,the href and rel attributes.

The < link > element links the webpage to the stylesheet

The href attribute is where the stylesheet is located.

The 'rel' attribute is the relationship between the wepbage and the stylesheet.


Internal stylesheet

The internal stylesheet is used to style the elements of one webpage.The internal stylesheet is written between the html < head > tag using the < style > element.


Inline style

The inline style is used to style single elements . To use the inline style , you simply put the style attribute to the html element you want to style. The style attribute can contain css property.
